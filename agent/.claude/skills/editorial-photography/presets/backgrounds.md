# Background Presets

Select a background preset to define the environment and setting for the generated images.

---

## Available Presets

### 1. Studio Grey (Default)

**Name:** `studio-grey`

**Prompt Snippet:**
```
seamless grey studio backdrop, clean and timeless, professional fashion studio environment
```

**Best For:** Any pose - most versatile option

---

### 2. Studio White

**Name:** `studio-white`

**Prompt Snippet:**
```
white cyclorama background, bright and commercial, high-key studio lighting environment
```

**Best For:** Any pose - commercial and clean aesthetic

---

### 3. Studio Black

**Name:** `studio-black`

**Prompt Snippet:**
```
dark void background, dramatic and moody, low-key studio with subject isolation
```

**Best For:** Editorial Drama, Confident Standing - dramatic impact

---

### 4. Industrial

**Name:** `industrial`

**Prompt Snippet:**
```
raw concrete walls with urban texture, exposed architectural elements, edgy industrial setting
```

**Best For:** Leaning Casual, Urban Lean, Editorial Drama

---

### 5. Warm Daylight

**Name:** `warm-daylight`

**Prompt Snippet:**
```
soft natural window light environment, warm and organic atmosphere, gentle diffused daylight
```

**Best For:** Relaxed Natural, Seated Editorial - intimate feel

---

### 6. Color Gel

**Name:** `color-gel`

**Prompt Snippet:**
```
bold solid colored backdrop with editorial gel lighting, vibrant and fashion-forward, saturated color environment
```

**Best For:** Editorial Drama, Confident Standing - high-fashion impact

---

### 7. Outdoor Urban

**Name:** `outdoor-urban`

**Prompt Snippet:**
```
city street environment, urban architectural context, real-world metropolitan setting with street elements
```

**Best For:** Street Walk, Urban Lean, Relaxed Natural

---

## Custom Background

If none of the presets fit, provide a custom background description following this format:

```
[environment type], [key visual elements], [atmosphere/mood]
```

**Example Custom:**
```
rooftop terrace at golden hour, city skyline in background with warm sunset tones, elevated urban luxury atmosphere
```

---

## Compatibility Guide

| Background | Recommended Poses |
|------------|-------------------|
| Studio Grey | Any (versatile) |
| Studio White | Any (versatile) |
| Studio Black | Editorial Drama, Confident Standing |
| Industrial | Leaning Casual, Urban Lean, Editorial Drama |
| Warm Daylight | Relaxed Natural, Seated Editorial |
| Color Gel | Editorial Drama, Confident Standing |
| Outdoor Urban | Street Walk, Urban Lean, Relaxed Natural |

---

## Usage

When generating images, the selected background preset snippet replaces `{BACKGROUND_PRESET_SNIPPET}` in the HERO_PROMPT template.
