## 1. Static Shots

### Static / Fixed Shot

| | |
|---|---|
| **Other Names** | Locked shot, Fixed frame, Locked off |
| **Equipment** | Tripod, C-stand, Apple boxes |
| **Feel** | Observational, tableau, formal, tension in stillness |

```
DIAGRAM: STATIC SHOT
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

                    [Subject]
                        │
                        │
                        │
                   ┌────┴────┐
                   │ [Camera]│  ← No movement
                   └────┬────┘
                        │
                   ───────────
                    (tripod)

    Camera remains completely stationary.
    Subject and environment may move within frame.
```

## 2. Pan Movements

### Pan Left / Pan Right

| | |
|---|---|
| **Other Names** | Horizontal pan, Swivel |
| **Equipment** | Tripod with fluid head, Pan handle |
| **Feel** | Surveying, following action, revealing space |

```
DIAGRAM: PAN LEFT / PAN RIGHT
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    PAN LEFT                          PAN RIGHT
    ←←←←←←←←                          →→→→→→→→

         ╲                                 ╱
          ╲                               ╱
           ╲                             ╱
       [Camera]                     [Camera]
           ●                             ●
           │                             │
       ─────────                     ─────────
       (tripod)                      (tripod)

    Camera ROTATES horizontally on a fixed pivot point.
    Base does NOT move. Only the camera head swivels.

    TOP VIEW:

              [Subject A]         [Subject B]
                   ╲                 ╱
                    ╲               ╱
                     ╲             ╱
                      ╲           ╱
                       ● [Camera]
                       │
                   (pivot point)
```

### Whip Pan / Swish Pan

| | |
|---|---|
| **Other Names** | Flash pan, Swipe, Zip pan, Blur pan |
| **Equipment** | Tripod with fluid head (loose tension), Handheld |
| **Feel** | Chaotic, energetic, time skip, transition |

```
DIAGRAM: WHIP PAN
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    MOTION BLUR DURING WHIP:

    Frame 1        Frame 2        Frame 3
    [Subject A]    ▓▓▓▓▓▓▓▓▓     [Subject B]
                   (blur)

    ════════════════════════════════════════════
    START          WHIP           END
    (clear)        (blur)         (clear)

    Speed: Extremely fast (creates motion blur)
    Often used to hide cuts or transition between scenes.
```

```
                   ╱
                  ╱
    [Camera] ●───●───● [Camera]
                  ╲
                   ╲
                    ● END (tilt down)

    Base does NOT move.
```

## 4. Roll Movements

### Dutch Angle / Dutch Tilt

| | |
|---|---|
| **Other Names** | Canted angle, Oblique angle, German angle, Batman angle |
| **Equipment** | Tripod with ball head, Handheld, Dutch head |
| **Feel** | Unease, tension, madness, stylized, off-kilter |

```
DIAGRAM: DUTCH ANGLE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    NORMAL FRAME:               DUTCH ANGLE:

    ┌─────────────┐             ╱╲
    │             │            ╱  ╲
    │  [Subject]  │           ╱    ╲
    │             │          ╱[Subj.]╲
    └─────────────┘         ╱        ╲
                           ╱──────────╲
    Horizon: ═══════       Horizon: ╲
    (level)                (tilted)

    Camera is rotated on its Z-axis (lens axis).
    The horizon line is no longer level.
```

---

### Roll / Barrel Roll

| | |
|---|---|
| **Other Names** | Camera roll, Rotation, Axial roll |
| **Equipment** | Gimbal, Specialized roll head, Drone |
| **Feel** | Disorientation, dreamlike, surreal, transition |

```
DIAGRAM: ROLL
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    Camera rotates around the lens axis:

    START           MID              END
    ┌────┐         ◢◣              ┌────┐
    │    │        ◢  ◣             │    │
    └────┘       ◢    ◣            └────┘
                ◥    ◤
    0°          ◥  ◤               360°
                 ◥◤
                 90°

    The entire frame rotates while camera points at same spot.
```

### Corkscrew

| | |
|---|---|
| **Other Names** | Spiral dolly, Roll + dolly |
| **Equipment** | Gimbal on dolly, Drone, Steadicam with roll |
| **Feel** | Surreal, hypnotic, dreamlike, disorienting |

```
DIAGRAM: CORKSCREW
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    Camera moves forward while rotating:

    TOP VIEW (showing roll):

    [Subject]
        │
        │    ◢◣
        │   ◢  ◣
        │  ●────●────●────● [Camera Path]
        │       ◥  ◤
        │        ◥◤
        │

    rolling (rotating on lens axis).

    Creates spiral/corkscrew visual effect.
```

## 5. Dolly Movements

### Dolly In / Push In

| | |
|---|---|
| **Other Names** | Push, Move in, Track in, Dolly forward |
| **Equipment** | Dolly + track, Slider, Steadicam, Gimbal, Wheelchair |
| **Feel** | Intimacy, focus, tension building, emphasis |

```
DIAGRAM: DOLLY IN / PUSH IN
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    TOP VIEW:

                    [Subject]
                        │
                        │
                        │
                        ▼
    [Camera START] ●────────────────● [Camera END]
                   ═══════════════════
                        (track)

    ──────────────────────────────────────────────

    SIDE VIEW:

    [Subject]           [Subject]
        │                   │
        │      ═══▶         │
    [Cam]               [Cam]
    START                END
    (far)              (close)

    Camera physically moves TOWARD subject.
    Background parallax shifts (unlike zoom).
```

```
    TRUCK LEFT ◀════════╪════════▶ TRUCK RIGHT
                        │
                   ═══════════════
                      (track)

    ──────────────────────────────────────────────

    TRUCK LEFT:                 TRUCK RIGHT:

    ┌─────────────┐             ┌─────────────┐
    │ →→→→→→→→→→→ │             │ ←←←←←←←←←←← │
    │  [Subject]  │             │  [Subject]  │
    │ →→→→→→→→→→→ │             │ ←←←←←←←←←←← │
    └─────────────┘             └─────────────┘
    Subject appears              Subject appears
    to move RIGHT               to move LEFT

    Camera moves laterally (sideways).
    Subject stays at constant distance.
```
